# 計算
滞在時間 = 探索時間 + 停止時間 + 体験時間 + 理解時間 + 写真時間

| 要素   | 対応軸               |
| ---- | ----------------- |
| 探索時間 | [[連続性]]           |
| 停止時間 | [[空間密度]]          |
| 体験時間 | [[活動性]]           |
| 理解時間 | [[物語可視性]]、[[時間層]] |
| 写真時間 | [[象徴性]]、[[記憶装置]]  |
# 要素
## 探索時間
観光客の基本動作は、歩く→見る→歩くと言える。
連続性が高いほど探索距離、すなわち時間が延びる。
## 停止時間
密度が高いと、探索中の止まる回数が増える。
## 体験時間
体験時間は滞在時間を直接増加させる。
## 理解時間
観光客は、意味・歴史・拝啓を理解すると滞在時間が延びる。
## 写真時間
現代観光では、写真やSNSが滞在時間を増やす。
