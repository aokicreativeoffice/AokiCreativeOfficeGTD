# 都市観察 Observation Hub

このHubは  
**都市フィールドワークで行う観察項目を整理する中心ノート**である。

Observationは

```
現地で何を見るか
```

を定義する。

---

# Observationの役割

都市研究では

```
Observation
↓
Method
↓
Framework
↓
Output
```

という知識の流れになる。

Observationは

- 現地観察
- フィールドノート
- 感覚記録

の基礎データになる。

---

flowchart TD
