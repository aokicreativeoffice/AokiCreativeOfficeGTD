# 規範構造

規範構造とは、社会において何が適切で何が逸脱かを定める期待とルールの構造である。

---

# 基本構造

```mermaid
graph TD

A[社会] --> B[期待]
B --> C[規範]
C --> D[行動統制]
```

---

# 規範の機能

- 行動予測可能性
- 協力の促進
- 秩序維持
- 逸脱の抑制

---

# 規範の種類

- 慣習
- 道徳
- 礼儀
- 法
- 組織規則
