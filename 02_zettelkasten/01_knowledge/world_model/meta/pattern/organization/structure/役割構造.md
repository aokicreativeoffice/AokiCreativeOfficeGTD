# 役割構造

役割構造とは、組織において各メンバーが担う機能や責任の分担構造である。

---

# 基本構造

```mermaid
graph TD

A[組織]

A --> B[経営]
A --> C[開発]
A --> D[営業]
A --> E[運営]
```

---

# 役割の機能

- 分業
- 専門化
- 効率化
