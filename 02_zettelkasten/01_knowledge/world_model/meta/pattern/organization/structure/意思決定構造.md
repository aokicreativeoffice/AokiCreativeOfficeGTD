# 意思決定構造

意思決定構造とは、組織において意思決定がどのような手順と権限で行われるかの構造である。

---

# 基本構造

```mermaid
graph TD

A[情報] --> B[分析]
B --> C[決定]
C --> D[実行]
```

---

# 意思決定の型

- トップダウン
- ボトムアップ
- 合議
