# 寡占形成パターン

市場では競争の結果、少数の企業が支配的地位を獲得する傾向がある。

---

# パターン構造

```mermaid
graph TD

A[多数企業競争] --> B[規模の経済]
B --> C[資本集中]
C --> D[企業淘汰]
D --> E[寡占市場]
```

---

# 典型的メカニズム

- 規模の経済
- ブランド力
- 技術優位
- 資本力
- ネットワーク効果

---

# 典型事例

- GAFA
- 日本の携帯キャリア
- 石油メジャー
