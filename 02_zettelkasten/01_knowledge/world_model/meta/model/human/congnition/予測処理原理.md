# 予測処理原理

人間の認知は、世界を予測し予測誤差を修正する過程である。

---

# 説明

人間の脳は受動的に情報を受け取るのではなく、

予測 → 観察 → 誤差 → 更新

という循環で世界を理解する。

つまり人間は常に

「次に何が起きるか」

を予測している。

---

# 含意

この原理は以下を説明する。

- 知覚
- 学習
- 期待
- バイアス
