# 最適化

## 定義

与えられた

- 目的
- 制約

の下で

**最も望ましい状態を見つける過程**

を **最適化（Optimization）** という。

---

# 基本構造

```mermaid
graph TD

Options --> Evaluation
Objective --> Evaluation
Constraints --> Evaluation
Evaluation --> Best
```

---

# 最適化の基本形

最適化問題は通常

```
目的関数
+
制約
```

で構成される。

例

```
利益 最大化
コスト 最小化
距離 最短化
時間 最小化
```
