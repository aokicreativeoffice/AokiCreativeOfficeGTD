# Summary
外集団（自分が属さない集団）に対して、その内部の個人差を過小評価し、均質であると認識してしまう認知バイアス。

---

# Description
人間は認知コスト削減のため、対象をカテゴリ単位で処理する。
このとき、

- 内集団 → 高解像度（個別認識）
- 外集団 → 低解像度（カテゴリ一括）

という非対称が発生する。

その結果、外集団は「みんな同じようなもの」と認識される。

---

# Structure
入力:
- 他者集団の情報（低頻度接触・断片的情報）

処理:
- カテゴリ化
- プロトタイプ抽出
- 情報圧縮

出力:
- 集団内差異の過小評価
- 単一イメージへの統合

---

# Dynamics / Mechanism
1. 接触頻度の差
   - 内集団 → 多数の個別経験
   - 外集団 → 少数の代表例のみ

2. 記憶の圧縮
   - 外集団はプロトタイプに集約される

3. 注意配分の偏り
   - 内集団は差異に注目
   - 外集団は共通点に注目

4. フィードバック
   - 単純化された認識がさらに観察を歪める

---

# Examples
- 「外国人はみんな同じに見える」
- ベトナム人＝グエン
- ロシア人＝イヴァン
- 「あの国の人はこういう性格」

---

# Implications
- ステレオタイプの形成
- 偏見・差別の温床
- 誤った意思決定（採用・交渉など）
