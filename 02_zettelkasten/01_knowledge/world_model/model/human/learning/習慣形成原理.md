# 習慣形成原理

繰り返された行動は自動化され習慣になる。

---

# 説明

人間の行動は

刺激 → 行動 → 報酬

の循環によって強化される。

この循環が繰り返されると

行動は意識的判断を必要としなくなる。

---

# 含意

この原理は

- 習慣
- 依存
- 行動パターン

を説明する。
