# 全体

```mermaid
flowchart TD

INPUT[User Input]

INPUT --> INTENT[Intent Interpretation<br>00_system]

INTENT --> TYPE{問いのタイプ判定}

TYPE -->|観察/事例| CASE_FLOW
TYPE -->|問題解決| SOLUTION_FLOW
TYPE -->|仮説検証| HYPOTHESIS_FLOW
TYPE -->|探索/理解| SEARCH_FLOW

CASE_FLOW --> OUTPUT
SOLUTION_FLOW --> OUTPUT
HYPOTHESIS_FLOW --> OUTPUT
SEARCH_FLOW --> OUTPUT

OUTPUT[Answer / Artifact]
```

#  このマップの本質
1. 問いを分類する → Intent
2. フローを選ぶ → Thinking Engine
3. グラフを辿る → Knowledge Graph
4. 知識を当てる → World Model
5. 答えを作る → Decision / Solution
