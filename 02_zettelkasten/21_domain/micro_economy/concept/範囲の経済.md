範囲の経済 (Economies of Scope)
​
# 1. 定義
​範囲の経済とは、複数の異なる製品やサービスを別々の企業が生産するよりも、一つの企業がまとめて生産する方が、総費用が低くなる現象を指す。
# 2. 数式表現
​2つの製品 X, Y を生産する費用を C とすると、以下の関係が成り立つ場合に範囲の経済が存在する。

C(X,Y)<C(X,0)+C(0,Y)

# 3. 発生の要因
​- 生産要素の共用: 同じ設備、土地、技術、データなどを複数の事業で使い回せる場合。
​- 副産物の活用: ある事業の廃棄物や余熱を、別の事業の原料やエネルギーとして利用できる場合。
​ブランドやノウハウの転用: 蓄積された管理能力や信頼性を新しい分野に活かせる場合。
​
# 4. 都市環境学・土木工学との関連
​- 多目的ダム: 「治水（洪水調節）」「利水（水道水）」「発電」を一つのダムで行うことは、範囲の経済の典型例である。
​- 道の駅・複合施設: 道路の「休憩機能」に「物産販売」や「防災拠点」を組み合わせることで、維持管理コストの効率化と地域振興を同時に図る。
​- 公共交通の多角化: 鉄道会社が駅ビル（商業）や不動産開発（住宅）を一体的に行うことで、交通需要の創出と経営の安定化を両立させる。
​- スマートシティ: センサーネットワークを「交通監視」「防災」「環境測定」など多目的に利用することで、デジタルインフラの投資効率を高める。
