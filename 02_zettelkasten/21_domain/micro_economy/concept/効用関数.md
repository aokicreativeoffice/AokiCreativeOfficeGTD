効用関数 (Utility Function)
# 1. 定義
​効用関数とは、消費者が財やサービスを消費することによって得られる「満足度（効用）」を、消費量の変数として数理的に表した関数である。通常、U(x, y) のように表記され、x や y は各財の消費量を指す。

# 2. 基本的な性質
​一般に、ミクロ経済学の基礎モデルでは以下の性質を仮定する。
​単調性（非飽和）: 消費量が増えれば増えるほど、効用も増加する（\partial U / \partial x > 0）。
​限界効用逓減の法則: 消費量が増えるにつれて、追加の1単位から得られる満足度の増加分（限界効用）は次第に小さくなる。
​凸性: 多様な財をバランスよく消費することを好む性質。

# 4. 都市環境学・土木工学との関連
​都市計画において、住民の「利便性」や「快適性」を定量化する際の基礎となる。
​交通手段選択: 所要時間や運賃を負の効用（費用）として捉え、どの交通手段が最も効用が高いかを分析する（離散選択モデルの基礎）。
​公共施設の配置: 公園や図書館へのアクセス性が、住民の効用関数にどう影響するかをモデル化し、最適な立地を検討する。
