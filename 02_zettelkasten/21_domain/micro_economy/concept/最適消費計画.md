最適消費計画 (Optimal Consumption Bundle)
# 1. 定義
​最適消費計画（または消費者均衡）とは、消費者が与えられた予算制約の下で、自らの効用を最大化する財の組み合わせを選択することである。
​
# 2. 均衡条件
​無差別曲線の傾き（限界代替率：MRS）と、予算制約線の傾き（価格比）が一致する点で達成される。
​数学的には以下の条件を満たす点である。

MRS_{xy} = \frac{MU_x}{MU_y} = \frac{P_x}{P_y}

# 3. 分析の応用
​代替効果: 価格変化により、相対的に安くなった財へ消費がシフトすること。
​所得効果: 価格変化により、実質的な購買力（所得）が変化すること。
# ​4. 都市環境学・土木工学との関連
​居住地選択モデル: 交通利便性と住環境（広さや静かさ）のトレードオフの中で、個人がどこに住むのが最も幸福（効用最大化）かを説明する。
​政策評価: インフラ整備による「時間短縮」が、個人の活動（仕事・余暇）の最適配分をどう変えるかを予測する。
