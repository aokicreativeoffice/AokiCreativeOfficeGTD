# エムス電報事件 (Ems Dispatch Incident)

## 1. Overview

1870年7月、プロイセン首相ビスマルクによる電報の改ざん・公表を契機として、フランス世論が激昂し普仏戦争へと至った外交事件。

本事件は、以下のRelationによって構造化される。

本図は、ビスマルクによる「情報の武器化（Context Stripping）」が、フランス宮廷内の「内部正当性の不安（Legitimacy Anxiety）」という構造的脆弱性に火をつけ、ナポレオン3世の合理的判断を封殺していくプロセスを示している。個人の病理（皇帝の結石）と組織の病理（急進的な世論依存）が、一つの電報によって同期された事例である。

  

Patterns

  

• [[The Prestige Trap]]: 権威が弱体化した組織は、実利よりも「面子（名誉）」を優先せざるを得ず、挑発に対して予測可能な暴走を見せる。

  

• [[Context Stripping]]: 事実を捏造するのではなく、文脈（丁寧な表現等）を削ぎ落とすことで、相手の感情的反応を制御する情報操作技術。

  

## 3. Analysis: 急進的体制の脆弱性

本事例は、急進的共和制や正当性の基盤が不安定な政体に対する以下の教訓を示す。

  

• Structural Weakness: 内部正当性に不安を抱えるシステムは、情報の微細な加工（Catalyst）によって容易に「感情的な全体主義」へと同期される。

  

• Incentive: 指導者層（ウジェニー等）にとって、戦争は国家の利益ではなく「体制の延命」というインセンティブで選択される。
