# ウィーン体制：正統主義と勢力均衡
## システムのメカニズム
- **五国同盟**: 英・露・墺・普・仏による「欧州の協調(Concert of Europe)」。
- **抑圧の論理**: 自由主義運動を「病」と見なし、共同で外科手術（軍事介入）を行う。
## メカニズム図

```mermaid
    Vienna --> Intervention[内部反乱への共同介入]
    Liberalism[自由主義/民族主義] -- "圧力" --> Vienna
```
