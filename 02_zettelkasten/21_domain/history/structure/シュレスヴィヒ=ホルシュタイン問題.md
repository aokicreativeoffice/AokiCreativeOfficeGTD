# シュレスヴィヒ・ホルシュタイン問題とデンマーク戦争

## 1. 概念定義 (Definition)
デンマーク王国の南部に位置する二つの公領（シュレスヴィヒとホルシュタイン）の帰属を巡る国際紛争。ビスマルクはこの「ナショナリズムの火種」を利用し、オーストリアを誘い出してデンマークを叩き、後の普墺戦争への布石とした。

## 2. 構造的背景：複雑な「パズル」
当時の英国首相パーマストンはこう語っている。
> 「この問題の複雑さを理解している者は3人しかいない。1人は死に（アルバート公）、1人は狂い（デンマークの学者）、もう1人は私だが、内容を忘れてしまった」

- **シュレスヴィヒ**: 北部。デンマーク人が多いが、ドイツ人も住む。
- **ホルシュタイン**: 南部。住民はほぼドイツ人。ドイツ連邦に加盟しているが、君主はデンマーク王。

## 4. 戦略のポイント：なぜオーストリアと組んだのか？

1. **ドイツ連邦の無力化**: 本来、この問題は「ドイツ連邦」全体の課題でしたが、ビスマルクはそれを無視し、オーストリアと二国間だけで解決した。これにより、小邦たちの影響力を排除した。
    
2. **オーストリアの孤立**: オーストリアは「ドイツ民族の保護者」という立場上、プロイセンの誘いを断れなかった。結果として、デンマークという「隣国」を敵に回し、さらに獲得した領土（ホルシュタイン）がプロイセンの中に孤立する「飛び地」となり、ビスマルクにいつでも揺さぶられる状態に陥った。

- `validates` [[ローン陸相の軍制改革]] (プロイセン軍の新型銃「ドライゼ銃」の初実戦テスト)

## 6. 考察：ビスマルクの「チェス」

ビスマルクはデンマーク戦争に勝つこと自体が目的ではなかった。彼の目的は、「オーストリアと一緒に仕事をし、その過程で相手の弱点を握り、いつでも喧嘩を売れる状態を作ること」にあった。 ガシュタイン協定で、わざと管理が難しい「飛び地」をオーストリアに押し付けたのは、将来の宣戦布告をあらかじめ「予約」する行為だった。
