# 正統主義 (Principle of Legitimacy)

## 1. 概念定義 (Definition)
1815年のウィーン会議において、フランス全権**タレーラン**が提唱した国際秩序の再建原則。「フランス革命以前（1789年以前）の正統な統治者が、正統な領土を支配する状態に戻すべきである」とする考え方。

## 2. 構造的メカニズム (Mechanism)

### A. 論理による防衛 (Logic as a Shield)
- **敗戦国フランスの救済**: フランスを「ナポレオンという暴徒に占領されていた被害者」と定義し直すことで、戦勝国による領土分割からフランス（ブルボン家）を守る武器となった。
- **革命の不当化**: 「国民の意志」による政権交代を「不当な簒奪（さんだつ）」と定義し、君主の権威を神聖不可侵なものとして再固定した。

### B. ウィーン体制の接着剤 (The Glue of Vienna)
- **現状維持 (Status Quo)**: 領土紛争を「元の持ち主に返す」というシンプルなルールで解決し、大国間の利害対立を一時的に凍結させた。
- **共同介入の正当化**: ある国で自由主義革命が起きた際、それを「正統な秩序への攻撃」と見なし、他国が軍事介入する法的根拠（神聖同盟・四国同盟）となった。



## 3. 主要な主体 (Key Actors)

| 主体 | 役割・インセンティブ |
| :--- | :--- |
| **タレーラン (仏)** | **提唱者**。敗戦国フランスの領土保全と、国際社会への早期復帰のためにこの論理を利用した。 |
| **メッテルニヒ (墺)** | **運用者**。多民族帝国オーストリアを維持するため、ナショナリズム（民族自決）を抑え込む「檻」として活用。 |
| **アレクサンドル1世 (露)** | **守護者**。キリスト教的友愛に基づく「神聖同盟」を提唱し、正統主義を宗教的権威で補強した。 |

## 4. 脆弱性と矛盾 (Vulnerabilities)
- **時間の不可逆性**: ナポレオンによって「自由」や「ナショナリズム」を植え付けられた民衆にとって、古い王政は「時代遅れのOS」であり、強い拒絶反応を引き起こした。
- **勢力均衡との衝突**: 「正統な領土」を返すことよりも、大国間の「勢力均衡（パワーバランス）」が優先されることが多く、ポーランドなどの小国はしばしば大国の犠牲となった。

## 6. 考察：タレーランの外交的勝利
正統主義は、一見すると「古臭い復古主義」に見えるが、その実態は**「敗戦国フランスが、戦勝国（英露墺普）を論理的に縛り付けた高度な外交カード」**であった。タレーランはこの論理を用いることで、フランスを「裁かれる側」から「秩序を作る側」へと一気に押し戻したのである。
