| 抽象行為   | 具体行為                 |
| ------ | -------------------- |
| 契約締結   | 乗車契約の成立（乗車時成立）       |
| 契約変更   | 行先変更 / 経路変更          |
| 契約解除   | 乗車拒否（正当事由）/ 途中降車     |
| 契約取消   | 誤乗車の訂正               |
| 債務履行   | 旅客の安全運送 / 指定地への輸送    |
| 債務不履行  | 運送不能 / 大幅遅延 / 安全義務違反 |
| 損害賠償   | 事故補償 / 運送事故責任        |
| 弁済     | 運賃返還 / 補償支払          |
| 権利移転   | 乗車権の譲渡（通常不可・例外的対応）   |
| 所有権移転  | 車内販売等（例外的）           |
| 債権譲渡   | 運賃債権の管理・回収委託         |
| 債務引受   | 配車委託・共同配車への引継ぎ       |
| 意思表示   | 乗車意思表示 / 配車依頼        |
| 意思表示取消 | 配車キャンセル              |
| 意思表示無効 | 誤配車 / 成立しない予約        |
| 占有     | 車両の占有管理              |
| 使用     | 車両運行 / 運転            |
| 収益     | 運賃収受 / 迎車料金          |
| 処分     | 配車枠管理 / 営業区域内運行      |
| 権利侵害   | 交通事故 / 安全配慮義務違反      |
| 損害発生   | 人身事故 / 物損事故          |
| 公示     | 運賃表示 / 営業許可表示        |ん
