| ------ | ---------------- |
| 契約締結   | ガイド契約 / 案内業務受託   |
| 契約変更   | 案内内容変更 / 言語変更    |
| 契約解除   | ガイドキャンセル         |
| 契約取消   | 誤契約取消            |
| 債務履行   | 観光案内 / 通訳 / 説明提供 |
| 債務不履行  | 案内未実施 / 不適切説明    |
| 損害賠償   | 誤案内 / 不適切情報提供    |
| 弁済     | 返金 / 補償          |
| 権利移転   | 案内契約の引継ぎ         |
| 所有権移転  | 資料提供（パンフ等）       |
| 債権譲渡   | ガイド報酬債権          |
| 債務引受   | 他ガイドへの引継ぎ        |
| 意思表示   | 案内受諾 / 見積提示      |
| 意思表示取消 | 案内辞退             |
| 意思表示無効 | 契約不成立            |
| 占有     | 資料・機材管理          |
| 使用     | 案内資料 / 翻訳ツール使用   |
| 収益     | ガイド報酬            |
| 処分     | 案内サービス提供         |
| 権利侵害   | 誤情報 / 名誉毀損       |
| 損害発生   | 誤案内による損害         |
| 公示     | ガイド資格表示（登録者）     |
