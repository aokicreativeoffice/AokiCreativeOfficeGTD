# Summary
出来事の原因・結果・構造を推定するエンジン

# Core Function
「なぜ起きたか」を説明する

# Inputs
- 事実
- 時系列
- 文脈

# Output
- 因果構造
- 説明モデル

# Thinking Protocol
1. 事象分解
2. 条件抽出（必要条件/十分条件）
3. 相互作用整理
4. 因果連鎖構築
5. 代替仮説比較

# Key Mechanisms
- フィードバック
- ロックイン
- 経路依存

# Constraints / Pitfalls
- 相関≠因果
- 単一原因化

# Examples
- 戦争の原因分析
