# Summary
ルール・規範・制度を解釈し、具体事例に適用するための推論エンジン

# Core Function
「何が許されるか／義務か」を決定する

# Inputs
- 条文・規約
- 事実関係
- 解釈原則

# Output
- 法的評価
- 適用結果

# Thinking Protocol
1. 条文構造分解（主体・要件・効果）
2. 文理解釈
3. 体系・目的解釈
4. 事実の当てはめ
5. 結論

# Key Mechanisms
- 解釈優先順位
- 規範階層
- 類推適用

# Constraints / Pitfalls
- 恣意的解釈
- 事実認定の誤り

# Examples
- 契約違反の判断
