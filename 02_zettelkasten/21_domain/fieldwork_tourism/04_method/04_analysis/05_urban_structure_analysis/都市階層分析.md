# 都市階層分析

## 概要

都市階層分析とは  
**都市内部の中心と周辺の階層構造を分析する方法**である。

都市には

- 中心
- 副中心
- 周辺

という階層が存在する。

---

# 都市階層の基本構造

```mermaid
flowchart TD

A[都市中心]
B[副中心]
C[周辺住宅地]
D[郊外]

A --> B
B --> C
C --> D
```
