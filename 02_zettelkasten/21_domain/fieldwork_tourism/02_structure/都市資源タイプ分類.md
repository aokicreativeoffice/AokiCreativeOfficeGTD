# Summary
都市の個性は資源の「量」ではなく「タイプと統合状態」によって決まる。本フレームは都市資源を機能的に分類し、個性形成への寄与を評価する。

---

# Core Concept

都市資源は以下の3タイプに分類される：

1. Core Asset（中核資源）  
2. Latent Asset（潜在資源）  
3. Generic Asset（汎用資源）  

---

# 1. Core Asset（中核資源）

## Definition
都市の構造・意味と強く結合し、非代替性が高い資源

## Characteristics
- 高い非代替性（Uniqueness）
- 空間構造と結合（Structure Integration）
- 歴史・宗教・地形などに基づく
- 長期的に持続

## Examples
- 氷川神社（大宮）
- 浅草寺（東京）
- 清水寺（京都）

## Evaluation

| 指標 | 評価 |
|------|------|
| Uniqueness | ◎ |
| Structure | ◎ |
| Meaning | ◎ |
| Visibility | ○（都市による） |

## Role
- 都市の「核」
- ナラティブの中心
- 回遊構造の起点

---

# 2. Latent Asset（潜在資源）

## Definition
高い価値を持つが、可視性や接続が弱く認知されていない資源

## Characteristics
- 非代替性は高い
- 意味は強いが露出が弱い
- 日常導線から外れている
- 知識依存性が高い

## Examples
- 盆栽町（大宮）
- 工業遺産
- 地元文化・職人技

## Evaluation

| 指標 | 評価 |
|------|------|
| Uniqueness | ◎ |
| Structure | ○ |
| Meaning | ◎ |
| Visibility | × |

## Role
- 都市の「深層」
- 強力な差別化ポテンシャル

## Problem
- 「存在しているが、存在していない扱い」

---

# 3. Generic Asset（汎用資源）

## Definition
どの都市にも存在し、差別化要因になりにくい資源

## Characteristics
- 全国的に分布
- 資本主導
- 高い可視性
- 高い再現性

## Examples
- 商業施設（ルミネ、イオン）
- チェーン飲食
- シネコン
- 吉本劇場

## Evaluation

| 指標 | 評価 |
|------|------|
| Uniqueness | × |
| Structure | △ |
| Meaning | △ |
| Visibility | ◎ |

## Role
- 集客装置
- 入口（Entry Point）
- 滞在時間の延長

---

# Structural Relationship

## 資源の相互関係

Core ──┐  
        ├─ 統合 → 都市個性  
Latent ─┘  

Generic → フック / 入口  

---

## 失敗パターン

Latent（強） × 非接続 → 埋没  
Generic（強） × 単独 → 均質化  

---

# Integration Model

都市個性は以下で決まる：

Urban Identity = Core × Latent × Integration

---

## Integrationの定義
- 導線で繋がっているか
- ナラティブで繋がっているか
- 体験として連続しているか

---

# Strategic Implications

## ① Latentの可視化
- 導線設計
- サイン・UI設計
- ナラティブ付与

---

## ② Genericの再定義
- 入口として利用
- Coreへの導線に変換

---

## ③ Coreの強化
- 回遊の中心に配置
- 意味の明確化

---

# Diagnostic Checklist

- Coreが存在するか？
- Latentが埋没していないか？
- Genericが支配していないか？
- 資源間の接続があるか？

---

# Application Example（大宮）

Core：氷川神社  
Latent：盆栽  
Generic：吉本・駅前商業  

→ LatentとCoreの接続不足  
→ Genericが入口を支配  

---

# Key Insight

都市の問題は資源不足ではなく、  
資源の「分類と接続の失敗」である
