# Summary
スーツの品質差は「寸法」ではなく「パターン・構造・縫製」によって決まる。
パターンオーダーやPittyはフィット調整の領域であり、ビスポークのみが構造設計まで介入可能。

---

# Core Idea

ジャケット品質は以下の4層で構成される：

① 寸法（サイズ）
② パターン（設計）
③ 構造（芯地・アームホール・いせこみ）
④ 縫製（立体形成）

→ 上位層ほど品質への影響が大きい  
→ ビスポークのみが全層を制御可能

# Dynamics / Mechanism

## 品質差の発生メカニズム

パターンオーダー：
既存パターン → 寸法変更  
→ フィット改善のみ

Pitty（店頭）：
既存パターン + 人的補正  
→ フィット + 見た目改善

ビスポーク：
型紙作成 → 仮縫い → 再設計  
→ 構造 + 立体 + 可動性

---

## 差の本質

```text
60→80点：
フィット・見た目の改善

80→100点：
構造・立体・可動性
```

# Evaluation

## 点数比較（ビスポーク＝100）

- ビスポーク：100
- Pitty（店頭）：75〜85
- Pitty（オンライン）：65〜75
- パターンオーダー：60〜75

---

## 点数の意味

- 70点：見た目は整う
- 80点：違和感が減る
- 90点：快適に動ける
- 100点：身体と一体化

---

# Implications

- ディテールでは品質は上がらない
- フィット改善だけでは上限がある
- 構造設計が品質の上限を決める

# Decision

## 意思決定フレーム

コスト重視 → パターンオーダー / Pitty  
バランス → Pitty（店頭）  
完成度重視 → ビスポーク
