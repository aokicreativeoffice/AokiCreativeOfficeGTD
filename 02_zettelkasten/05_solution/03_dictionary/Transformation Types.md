# ルーティング辞書（最重要）
|歪み|変換|
|---|---|
|不足 | 増加 / 強化 / 集約|
|過剰 | 削減 / 制約 / フィルタ|
|非対称 | 可視化 / シグナル / 共有|
|断絶 | 接続 / ブリッジ / 再配線|
|過密 | 分散 / 時間分割 / 上限設定|
|ロックイン | 分離 / 再配線 / 置換|
|暴走 | フィードバック / 制御 / 緩衝|
|低効率 | 最適化 / 標準化 / 自動化|
|低品質 | フィルタ / 選別 / 教育|
|不信 | 可視化 / 信頼構築 / 契約|

---

# 使い方
1. 問題を設定する
2. その問題の歪みを同定する
3. その歪みに対応する変換を見る
4. 具体的なSolutionを生成する
