# Problem Type Routing Template

## ■ explanation（解説）

### ■ 出力構造
- Concept
- Structure
- Mechanism（5フェーズ必須）
- Relation（5以上）
- Pattern
- Implication
- 一行圧縮

### ■ 特記事項
- Solution / Decisionは禁止
- 転換点を必ず含める

---

## ■ analysis（原因分析）

### ■ 出力構造
- 問題定義
- 表層原因
- 中間原因
- 深層構造
- 転換点
- 反証
- Pattern

### ■ 特記事項
- 単一原因禁止
- 因果3層必須

---

## ■ decision（意思決定）

### ■ Entry Hubフル実行

1. Pattern列挙
2. Solution（3案以上）
3. Instance
4. Decision（Scoring）
5. Execution

---

## ■ design（設計）

### ■ 出力構造
- 要件定義
- 制約
- アーキテクチャ
- コンポーネント
- フロー
- リスク
- 拡張性

---

## ■ Routingルール

- explanation → Entry Hub停止
