# 概要
観光地では、観光圧力と生活空間のバランスによって  都市構造が変化する。
観光地は観光 vs 生活の力学によって形成される。

# 基本モデル
観光地の構造は、観光圧力 ÷ 生活密度で決まる。

|状態|構造|
|---|---|
|観光 << 生活|生活都市|
|観光 ≈ 生活|共存型観光|
|観光 > 生活|観光化|
|観光 >> 生活|完全観光都市|

# 観光圧力
観光圧力とは、観光客数 + 観光資本 + 観光消費によって生じる都市変化の力。

主な指標
- 年間観光客数
- 観光密度
- 民泊数
- 観光店舗割合
- 観光投資
# 生活密度
生活密度とは、居住人口 + 生活機能 + 地域コミュニティの強さ。

指標
- 居住人口
- 学校
- 地元商店
- 住宅比率
# 力学の結果
## 1 生活優勢
観光圧力 < 生活密度

観光は補助的機能。

例
- 地方都市
- 小規模観光地
## 2 共存型観光地
観光圧力 ≈ 生活密度

観光と生活が共存。

例
- 尾道
- 金沢

特徴
- 生活空間が残る
- 観光空間が限定
## 3 観光化
観光圧力 > 生活密度

生活機能が弱まり  、観光機能が拡大。

特徴
- 店舗観光化
- 民泊増加
- 住宅減少

例
- 京都中心部
## 4 完全観光都市
観光圧力 >> 生活密度

都市機能が、観光専用になる。

例
- ベネチア
- テーマパーク都市
# 観光化のメカニズム
観光圧力が増えると、地価上昇 ・住宅減少 ・店舗化が進み観光化する。
この現象は、Tourism Gentrificationと呼ばれる。
# 空間構造の影響
都市構造によって観光圧力の影響は変わる。
## 開放型都市

- 平地・道路網
- 観光拡散

例
- 京都
- 奈良

観光化しやすい。



## 制約型都市

- 地形制約
- 観光集中

例
- 尾道
- 松本

生活空間が守られる。
# 観光タイプの影響
観光タイプによって観光圧力は異なる。

|観光|圧力|
|---|---|
|ブランド観光|高|
|テーマ観光|高|
|探索観光|中|
|自然観光|低|

# 観光地の安定条件
観光地が持続する条件
観光圧力 < 生活密度 または 観光空間と生活空間の分離
# 観光OSとの関係
観光地評価は、魅力 × 探索成功率 × 観光圧力で理解できる。
観光圧力が過剰な場合、オーバーツーリズムが発生する。
# メモ
観光都市の設計では、観光・生活・空間構造のバランスが重要である。
