# Value Proposition（価値提案モデル）

Value Proposition（価値提案）とは

**顧客の問題を解決し  
顧客に具体的な価値を提供する約束**

である。

価値提案は、
1. 顧客 
2. 問題 
3. 解決 
4. 価値
の構造で設計される。

---

# 価値提案の基本構造

1. Customer
2. Problem
3. Solution
4. Value
営業・マーケティング・商品設計は  
この構造に基づいて設計する。

---

# 価値提案の式

価値提案は次の式で表せる。
Value = Problem Solving + Benefit
顧客は、価値 > 費用の場合のみ購入する。

---

# 価値提案の3要素

## 1 Customer（顧客）

誰のための価値か。

例
- 地方バス会社
- 観光会社
- 物流企業
- 自治体

---

## 2 Problem（問題）

顧客が抱える問題。

例
- 人手不足
- 業務属人化
- 監査対応
- DX遅れ

良い問題の条件
- 頻繁に発生する
- 深刻である
- 解決ニーズが強い

---

## 3 Solution（解決）

問題を解決する方法。

例
- 業務OS
- SaaS
- コンサル
- 外注

---

# 価値の種類

顧客価値は次の4種類に分類できる。
- 収益増加
- コスト削減
- リスク低減 
- 時間削減
---

# 価値提案の例

例

|要素|例|
|---|---|
|顧客|地方バス会社|
|問題|監査対応が複雑|
|解決|監査チェックシステム|
|価値|行政リスク低減・業務効率化|

# Value Proposition Canvas

価値提案は次の対応関係で設計する。

| 顧客    | 価値   |
| ----- | ---- |
| 業務    | 製品   |
| 困りごと  | 問題解決 |
| 得たい成果 | 価値創出 |
---

# 良い価値提案

良い価値提案の条件

- 顧客問題に直結
- 明確な利益
- 実行可能
- 競争優位

---

# 悪い価値提案

よくある失敗
- 機能説明
顧客は機能ではなく、価値を購入する。

---

# 価値提案のテスト

価値提案は営業で検証する。
1. 仮説 
2. 顧客接触
3. 提案
4. 顧客反応 
5. 改善

---

# まとめ

価値提案とは、顧客から問題を発見し、それを解決することで生まれる価値を明確に定義することである。
事業成功の核心は 顧客問題 × 価値 の適合である。
