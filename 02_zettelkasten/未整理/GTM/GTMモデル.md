# GTM（Go-To-Market）モデル

GTMとは、市場に対して、どの顧客に、どの価値を  、どの方法で届けるかを設計する戦略である。

---

# GTM構造
1. Market
2. Segment
3. ICP
4. Value Proposition
5. Channel
6. Sales Model
7. Revenue Model

---

# 1 Market（市場）

どの市場を攻めるか。

| 評価軸  | 評価  |
| ---- | --- |
| 市場規模 |     |
| 問題強度 |     |
| 競争   |     |

---

# 2 Segment（顧客セグメント）

市場を分解する。

例
運輸
│ 
├バス 
├タクシー 
├物流 
└観光

---

# 3 ICP（理想顧客）

最も成功確率の高い顧客。

例
- 地方バス会社 
- 車両20〜80台 
- 社長決裁

---

# 4 Value Proposition（価値）

顧客に提供する価値。

構造
1. 顧客問題
2. 解決
3. 価値

例
1. 監査の対応がある／対応が難しい
2. 業務OSで対応を楽にする
3. 行政リスクを削減する

---

# 5 Channel（チャネル）

顧客にどう届けるか。

例
- 営業
- 紹介
- 代理店
- 展示会
- オンライン

---

# 6 Sales Model（営業モデル）

営業方法。

例
- 直販営業
- インサイドセールス
- コンサル営業

---

# 7 Revenue Model（収益モデル）

収益の仕組み。

例
- 導入費
- サブスク
- コンサル
- ライセンス
コードをコピーする

# GTMの評価

GTMの良し悪しは、顧客適合 × 価値適合 × チャネル適合 で決まる。
