# Japan World Model

Japan World Model は、日本文化が成立する社会構造を説明するモデルである。

このモデルは

- 地理
- 経済
- 政治
- 宗教
- 社会
- 美意識

などの構造から日本文化を理解する。

# World Model と Kernel

World Model は文化の環境であり、Kernel は文化の原理である。

```mermaid
graph TD
WorldModel["World Model"]
Kernel["Kernel"]
Culture["Culture"]

WorldModel --> Kernel
Kernel --> Culture
```
