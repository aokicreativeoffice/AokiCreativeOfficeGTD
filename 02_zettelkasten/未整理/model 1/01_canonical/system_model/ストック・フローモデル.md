ストック・フローモデルとは、システム内の蓄積量（ストック）と、その増減を生む流れ（フロー）の関係を表すモデルである。
# Translation
stock and flow

# Engine
## 要素
- ストック
- 流入
- 流出
## 構造

```mermaid
flowchart LR

Inflow[Inflow<br>流入]
Stock[Stock<br>蓄積]
Outflow[Outflow<br>流出]

Inflow --> Stock
Stock --> Outflow
```
ストックは、流入と流出によって変化する。
# Understanding
ストック・フローモデルは
- [[12 システム]]
- [[10 効率]]    
- [[因果]]   
の理解に役立つ。
多くの社会現象は、ストックとフローによって説明できる。
# Background
ストック・フロー概念は
- 経済学
- システムダイナミクス
- 生態学
などで発展した。
システムの変化は、蓄積量の変化として理解できる。
# Example
人口
```mermaid
flowchart LR

Birth[出生] --> Population[人口]
Population --> Death[死亡]
```
# Use
- 人口分析
- 経済分析
- 在庫管理
- 資源管理
