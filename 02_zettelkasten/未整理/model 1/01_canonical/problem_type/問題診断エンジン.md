# Problem Diagnosis Engine

Problem Diagnosis Engine は、
観察された現象を Problem Type に分類するための診断構造である。

問題を正しく分類することで、
適切な分析方法を選択できる。

---

# 診断フロー

Observation
↓
Problem Definition
↓
Problem Diagnosis
↓
Problem Type
↓
Analysis Method
