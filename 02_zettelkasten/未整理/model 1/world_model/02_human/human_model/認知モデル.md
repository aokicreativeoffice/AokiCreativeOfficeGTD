# Cognition Model（認知モデル）

認知とは、知覚された情報を処理し、意味・概念・判断を形成する過程である。
知覚が「入力」であるのに対し、認知は情報処理と意味構築の段階である。

---

# 認知の基本構造

### 1. Perception（知覚）

外界の刺激が注意によって選択され、知覚される。

入力情報
- 視覚    
- 聴覚    
- 言語    
- 社会情報    

---

### 2. Encoding（符号化）

知覚された情報が内部表現に変換される。
ここで、
- パターン認識    
- 概念化    
が行われる。

例
- 線と円 → 顔    
- 音声 → 言語    

---

### 3. Memory Activation（記憶活性）

既存の記憶や知識が呼び出される。
参照されるもの
- 過去経験    
- スキーマ    
- 概念知識    
- 文化    

ここで意味ネットワークが働く。

---

### 4. Interpretation（解釈）

情報の意味が決定される。
影響要因
- 文脈    
- 期待    
- 価値観    
- 感情

この段階で
- [[02_zettelkasten/01_knowledge/world_model/meta/model/human/congnition/認知バイアス]]    
- [[02_zettelkasten/01_knowledge/world_model/model/human/congnition/フレーミング効果]]    
が発生する。

---

### 5. Belief Update（信念更新）

新しい情報に基づき、
- [[信念]]    
- [[世界モデル]]   
が更新される。

例
- 「この人は信頼できる」    
- 「この場所は危険」    

---

### 6. Judgment（判断）

評価や判断が形成される。

例
- 良い / 悪い    
- 有利 / 不利    
- 安全 / 危険    

この判断が意思決定に渡される。

---

# 認知処理の全体フロー
1. 知覚
2. 符号化
3. 記憶活性化
4. 解釈
5. 信念更新
6. 判断

---

# 認知の主要機能

## 概念化

情報をカテゴリに分類する。

例
- 動物    
- 食べ物    
- 危険    

---

## 推論

既存知識から結論を導く。

例
- 演繹    
- 帰納    
- アブダクション    

---

## 問題解決

目標達成のための思考。

例
- 計画    
- 仮説    
- 試行    

---

# 認知の制約

認知は完全ではない。

主な制約
- 認知負荷    
- 情報不足    
- 時間制約    
- 記憶容量    

その結果
- [[02_zettelkasten/01_knowledge/world_model/model/human/congnition/ヒューリスティック構造]]
- [[02_zettelkasten/01_knowledge/world_model/meta/model/human/congnition/認知バイアス]]

が生じる。

# 認知と感情

感情は認知に影響する。
1. 感情
2. 情報処理
3. 解釈

例
- 不安 → 危険過大評価    
- 喜び → 楽観的判断

# 応用

顧客の意思決定は
1. 情報接触（広告・口コミ）
2. 認知（ブランド理解）
3. 評価（価値判断）
4. 購買（意思決定）

---

1. 知覚（景観・文化）
2. 認知（意味理解）
3. 感情（感動）
4. 記憶（旅行体験）
