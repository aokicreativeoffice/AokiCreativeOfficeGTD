# 生成ルール

1. concept_typeは1つ以上（主型1つ）
2. relationsは任意（なければ削除）
3. Summaryは必ず書く
4. Examplesは最低1つ
5. Implicationsで「使い道」を書く
6. 下記からノートの種類別にテンプレートを当て嵌める
