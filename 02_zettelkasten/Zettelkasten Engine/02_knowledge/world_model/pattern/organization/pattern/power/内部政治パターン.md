# 内部政治パターン

組織内部で権力や資源を巡って政治的行動が行われる現象。

意思決定が合理性ではなく権力関係によって左右される。

---

# パターン構造

```mermaid
graph TD

A[組織]

A --> B[権力争い]

B --> C[派閥形成]

C --> D[政治的意思決定]
```

---

# 発生要因

- 権力競争
- 資源配分
- 昇進争い

---

# 結果

- 非合理意思決定
- 組織分裂
- 効率低下

---

# 例

- 政党内部政治
- 企業内派閥争い
