# 臨界点パターン

システムは徐々に変化しているように見えても、  
ある閾値を超えると急激な変化が発生する。

この閾値を臨界点と呼ぶ。

---

# パターン構造

```mermaid
graph TD

A[徐々な変化] --> B[蓄積]
B --> C[臨界点到達]
C --> D[急激変化]
```

---

# 例

- 社会革命
- 市場暴落
- 生態系崩壊
