# ロックインパターン

初期条件や偶然の選択が繰り返し強化され、  
他の選択肢が排除されて固定される現象。

---

# パターン構造

```mermaid
graph TD

A[初期選択] --> B[利用増加]
B --> C[環境適応]
C --> D[選択固定]
D --> B
```

---

# 例

- QWERTY配列
- OS市場
- 規格競争
