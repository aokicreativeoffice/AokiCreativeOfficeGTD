# 集団構造

集団構造とは、複数の個人が共通の目的・所属・関係によってまとまる構造である。

---

# 基本構造

```mermaid
graph TD

A[個人群] --> B[集団形成]
B --> C[内集団]
B --> D[外集団]
```

---

# 集団の要素

- 所属
- 境界
- 役割
- 規範
- アイデンティティ

---

# 集団の種類

- 家族
- 地域共同体
- 階級
- 宗教集団
- ファンダム
- nation
