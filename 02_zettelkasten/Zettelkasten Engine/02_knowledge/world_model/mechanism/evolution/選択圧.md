# 選択圧

## 定義

ある環境条件・制約・相互作用によって  
特定の形質・行動・戦略が

**有利または不利になる圧力**

を **選択圧（Selection Pressure）** という。

# 選択圧の本質

選択圧とは

```
ある特徴が
有利になるか
不利になるか
```

を決める環境条件である。

# 選択圧と進化

進化は

```
変異
+
選択圧
```

によって起こる。

選択圧が変わると  
進化の方向も変わる。

---

# 選択圧と適応

適応とは

```
選択圧への対応
```

である。

# 要約

選択圧とは

**特定の形質・行動・戦略を有利または不利にする環境条件**

であり、  
進化・適応・淘汰を生むメカニズムである。
