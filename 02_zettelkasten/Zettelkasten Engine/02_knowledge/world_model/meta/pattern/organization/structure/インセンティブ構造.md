# インセンティブ構造

インセンティブ構造とは、組織メンバーの行動を動機づける報酬・評価・罰則の仕組みである。

---

# 基本構造

```mermaid
graph TD

A[行動] --> B[評価]
B --> C[報酬]
```

# インセンティブの種類

- 金銭報酬
- 地位
- 評価
- 承認
