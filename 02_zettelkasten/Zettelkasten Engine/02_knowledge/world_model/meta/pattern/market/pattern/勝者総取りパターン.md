# 勝者総取りパターン

ネットワーク効果や規模の経済によって、1つの企業が市場の大部分を占有するパターン。

---

# パターン構造

```mermaid
graph TD

A[ユーザー増加] --> B[価値増加]
B --> C[さらにユーザー増加]
C --> D[競争優位]
D --> E[市場支配]
```

---

# 説明

ユーザーが増えるほど価値が上がる市場では、最も大きい企業がさらに有利になる。

その結果

- 競争が急速に収束
- 巨大企業が出現

する。

---

# 例

- Google
- Facebook
- Windows
