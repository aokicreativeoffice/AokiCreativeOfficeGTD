# パターン認識原理

人間の認知は、環境の中に繰り返し現れるパターンを検出することで成立する。

---

# 説明

人間は世界を個別の出来事としてではなく、  
共通する特徴を持つパターンとして認識する。

この能力により

- 物体認識
- 言語理解
- 状況判断
- 問題解決

が可能になる。

パターン認識は

観察 → 類似性検出 → 抽象化

という過程で行われる。

---

# 含意

この原理は以下を説明する。

- 概念形成
- 分類
- 直感
- 専門技能

専門家は大量の経験により  
高度なパターン認識能力を持つ。

---

# 例

例

医師  
→ 症状のパターンを認識

チェスプレイヤー  
→ 局面パターンを認識

エンジニア  
→ 故障パターンを認識
