# 目標志向原理

人間の行動は目標に向けて組織される。

---

# 説明

人間は未来の状態を想定し、

現在の行動を調整する。

行動は

目標 → 計画 → 実行 → 評価

の循環で進む。

---

# 含意

この原理は

- 計画
- 努力
- 戦略

を説明する。
