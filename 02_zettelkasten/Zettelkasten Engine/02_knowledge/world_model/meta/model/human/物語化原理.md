# 物語化原理

人間は出来事を物語として理解する。

---

# 説明

人間は出来事を単なる事実としてではなく、

- 原因
- 結果
- 主人公
- 目的

を持つ物語として理解する。

---

# 含意

この原理は

- 歴史理解
- 自己理解
- 文化

に影響する。
