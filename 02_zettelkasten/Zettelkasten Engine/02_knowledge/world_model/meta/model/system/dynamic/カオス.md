# カオス

決定論的なシステムであっても、初期条件のわずかな違いによって長期的な挙動が大きく変化する現象。

---

# 説明

カオスとは、システムの振る舞いが完全にランダムではないにもかかわらず、
長期的に予測が非常に困難になる現象である。

カオス系では、初期条件の小さな差が時間とともに拡大する。

これを **初期値鋭敏性** と呼ぶ。

---

# 基本構造

初期条件の差  
↓  
時間経過  
↓  
差の増幅  
↓  
予測困難

---

# 例

天気

わずかな条件差  
↓  
数日後に大きな違い

---

# 含意

カオスは

- 気象
- 生態系
- 経済
- 社会

など多くの複雑系で観察される。
