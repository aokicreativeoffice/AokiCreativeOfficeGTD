# 3 バイパス型モデル（フランス型）

都市中心を高速線がバイパスし、郊外に高速駅を設置するモデル。

## 構造
旧都市駅
│
│支線
│
高速バイパス ──── 高速駅
コードをコピーする

## 特徴

- 都市中心駅と高速駅が分離
- 都市を高速線が回避

## メリット

- 高速性能を維持
- 都市アクセスも維持

## デメリット

- 駅が複数になる
- 都市機能が分散

## 代表例

- TGV
