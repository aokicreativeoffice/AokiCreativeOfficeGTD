# 適応

## 定義

主体・生物・組織・システムが  
環境条件や制約に対して

**より生存・存続しやすい形へ変化する過程**

を **適応（Adaptation）** という。

---

# 基本構造

```mermaid
graph TD

Environment["環境"]
Variation["変異・多様性"]
Selection["選択"]
Adaptation["適応"]

Environment --> Selection
Variation --> Selection
Selection --> Adaptation
```

つまり

```
環境
+
多様性
+
選択
↓
適応
```

である。

---

# 適応の本質

## 1 環境との適合

適応とは

```
環境条件
↓
生存に有利
```

となる変化である。

---

## 2 完全ではない

適応は

```
最適
```

ではなく

```
十分に生存可能
```

な状態である。

---

## 3 動的である

環境が変われば

```
適応
↓
再適応
```

が必要になる。

# 適応と進化

進化は

```
変異
+
選択
+
適応
```

によって進む。

---

# 適応と制約

適応は

```
制約の中で
最も生存可能な形
```

を作る。

# 要約

適応とは

**環境条件や制約の中で生存・存続に有利な形へ変化する過程**

であり、  
進化やシステム変化の基本原理である。
