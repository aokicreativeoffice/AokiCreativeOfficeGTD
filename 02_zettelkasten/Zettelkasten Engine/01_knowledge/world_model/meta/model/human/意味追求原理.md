# 意味追求原理

人間は自分の経験や行動に意味を与えようとする。

---

# 説明

人間は単に行動するだけでなく、  
その行動や出来事を解釈し意味づけする。

意味づけは

- 価値観
- 物語
- アイデンティティ

と結びつく。

---

# 含意

この原理は

- 宗教
- 文化
- 人生観

の形成を説明する。
