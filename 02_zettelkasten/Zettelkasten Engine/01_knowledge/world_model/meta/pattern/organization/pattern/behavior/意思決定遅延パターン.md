# 意思決定遅延パターン

組織内の意思決定が遅くなり、環境変化への対応が遅れる現象。

---

# パターン構造

```mermaid
graph TD

A[問題発生] --> B[情報収集]
B --> C[承認プロセス]
C --> D[追加検討]
D --> E[決定遅延]
```

---

# 発生要因

- 階層構造
- 官僚制
- 承認プロセス
- 情報不足

---

# 結果

- 市場機会喪失
- 危機対応失敗
- 組織競争力低下

---

# 例

- 大企業の新規事業失敗
- 政府の危機対応遅延
