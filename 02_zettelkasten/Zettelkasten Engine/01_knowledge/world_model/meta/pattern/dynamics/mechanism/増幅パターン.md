# 増幅パターン

小さな変化が連鎖的に強化され、大きな変化に発展するダイナミクスを増幅パターンと呼ぶ。

---

# パターン構造

```mermaid
graph TD

A[小さな変化] --> B[影響拡大]
B --> C[さらなる変化]
C --> D[影響増幅]
D --> B
```

---

# 例

- SNS拡散
- 投資バブル
- 人気商品
