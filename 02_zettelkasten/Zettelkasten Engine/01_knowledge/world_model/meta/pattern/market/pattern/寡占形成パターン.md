# 寡占形成パターン

競争市場において、時間の経過とともに少数企業へ市場シェアが集中し、寡占構造へ移行する典型的な市場パターン。

---

# パターン構造

```mermaid
graph TD

A[多数企業競争] --> B[競争淘汰]
B --> C[企業退出]
C --> D[市場集中]
D --> E[寡占形成]
```

---

# 説明

競争市場では、すべての企業が長期的に生き残るわけではない。

競争を通じて

- 技術力
- 資本力
- ブランド
- 規模

などで優位な企業が生き残り、企業数は徐々に減少する。

---

# 発生要因

## 規模の経済

大企業ほどコストが低くなる。

## ブランド優位

信頼が集中する。

## 技術格差

研究開発力の差。

## 参入障壁

新規参入が困難。

---

# 例

- 自動車産業
- 半導体産業
- 航空産業
