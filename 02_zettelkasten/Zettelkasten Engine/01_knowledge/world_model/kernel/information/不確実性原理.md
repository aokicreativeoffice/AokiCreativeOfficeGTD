# 不確実性原理

## 定義

**不確実性とは、未来の状態や結果が確定していない状態である。**

主体は

```
未知
不完全情報
ランダム性
```

の中で意思決定を行う。

---

## 基本命題

### 1 完全情報は存在しない

主体は常に

```
部分情報
```

で判断する。

---

### 2 不確実性は探索を生む

未来が分からないため

```
探索
試行
学習
```

が行われる。

---

### 3 不確実性はリスクを生む

意思決定には

```
成功
失敗
```

の可能性がある。

---

### 4 情報は不確実性を減少させる

情報取得によって

```
予測精度
```

が改善する。

---

## 典型構造

```
不確実性
↓
探索
↓
学習
↓
適応
```
