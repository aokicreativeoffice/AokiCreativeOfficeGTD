# 情報原理

## 定義

**情報とは、不確実性を減少させる差異である。**

主体が世界について持つ不確実性が減少するとき、その差異を情報という。

---

## 基本命題

### 1 不確実性減少

情報は **不確実性を減少させる。**

例

- 天気予報
- 地図
- 標識
- 口コミ

すべて

```
未来の状態の予測精度
```

を改善する。

---

### 2 情報は差異である

情報は **変化や差異として現れる。**

例

- 価格差
- ニュース
- 異常値
- 信号

差が存在しなければ情報は存在しない。

---

### 3 情報は物理媒体を必要とする

情報は必ず

```
物理媒体
```

に保存される。

例

- 文字
- 音
- 光
- 電気信号
- DNA

---

### 4 情報は伝播する

情報は

```
主体 → 主体
```

へ伝播する。

伝播により

- 評判
- 流行
- 知識
- 噂

が生まれる。

---

### 5 情報は価値を持つ

情報は

```
意思決定の改善
```

によって価値を持つ。

例

- 市場価格
- 技術知識
- 地理情報
- 軍事情報

---

## 数理表現

情報量

```
I = -log2(p)
```

p  
= 事象の確率

確率が低いほど

```
情報量は大きい
```

---

```
不確実性
     ↓

     ↓
知識更新
     ↓
意思決定
```
