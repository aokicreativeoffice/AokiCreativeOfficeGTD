# 認知原理

## 定義

**認知とは、主体が外界の情報を取得し、解釈し、内部モデルとして構築する過程である。**

人間や生物は世界を直接理解するのではなく

```
知覚
↓
解釈
↓
内部モデル
```

によって世界を認識する。

---

## 基本命題

### 1 認知は世界の写像ではない

主体は

```
世界そのもの
```

を認識するのではなく

```
内部モデル
```

を構築する。

つまり認知は

```
現実
→ 主観的表象
```

の変換である。

---

### 2 認知は選択的である

主体は

```
すべての情報
```

を処理できない。

そのため

```
注意
```

によって情報を選択する。

これを

```
注意資源制約
```

という。

---

### 3 認知は予測的である

主体は

```
入力
```

を受け取るだけではなく

```
予測
```

を行う。

現代認知科学では

```
予測処理
```

として説明される。

---

### 4 認知は学習によって更新される

内部モデルは

```
経験
↓
学習
↓
更新
```

される。

---

### 5 認知は意味づけを行う

人間は情報を

```
意味
物語
カテゴリー
```

として整理する。

これによって

```
理解
判断
行動
```

が可能になる。

---

## 典型構造

```
環境
↓

内部モデル

行動
```
