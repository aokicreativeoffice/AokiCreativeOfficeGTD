# 時間原理

## 定義

**時間とは、状態の変化が順序を持って積み重なる構造である。**

時間は

- 変化
- 順序
- 累積

によって構成される。

---

## 基本命題

### 1 時間は順序を持つ

出来事は

```
過去 → 現在 → 未来
```

という順序で並ぶ。

この順序は因果関係を成立させる基盤となる。

---

### 2 多くの現象は不可逆である

多くの現象は

```
未来 → 過去
```

には戻らない。

例

- 歴史
- 成長
- 老化
- 技術進歩

この不可逆性は多くの場合  
[[02_zettelkasten/Zettelkasten Engine/01_knowledge/world_model/kernel/physics/エントロピー増大原理]]と関連する。

---

### 3 過去は現在を制約する

現在の状態は

```
過去の状態
+
過去の選択
```

によって決まる。

これを

```
履歴依存
```

という。

---

### 4 変化は累積する

多くの現象では

```
小さな変化
↓
蓄積
↓
構造変化
```

が起こる。

例

- 都市成長
- 文化変化
- 技術進化

---

### 5 時間はリズムや周期を生む

多くの現象は

- 周期
- サイクル
- 季節

を持つ。

例

- 景気循環
- 季節観光
- 生態周期

---

## 典型構造

```
状態
↓
変化
↓
累積
↓
歴史
```
