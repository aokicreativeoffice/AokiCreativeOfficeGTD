# 権力集中パターン

組織の意思決定権が少数の人物または中心組織に集中するパターン。

---

# パターン構造

```mermaid
graph TD

A[組織] --> B[権限委任]
B --> C[中心権力]
C --> D[意思決定集中]
```

---

# 発生要因

- 組織成長
- 危機対応
- カリスマリーダー
- 官僚化

---

# 例

- CEO中心経営
- 強い大統領制
- 軍司令部集中
