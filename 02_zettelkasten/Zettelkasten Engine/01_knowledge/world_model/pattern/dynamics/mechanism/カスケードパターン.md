# カスケードパターン

ある変化が次々と他の要素に伝播し、連鎖的にシステム全体へ広がる現象をカスケードと呼ぶ。

---

# パターン構造

```mermaid
graph TD

A[初期変化] --> B[局所影響]
B --> C[連鎖発生]
C --> D[拡大]
D --> E[システム全体変化]
```

---

# 例

- 金融危機
- 停電連鎖
- 銀行破綻
