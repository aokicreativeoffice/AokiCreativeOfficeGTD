# 分極化パターン

社会の意見や集団が中間を失い、相互に遠ざかって対立的な二極へ分かれていくパターン。

---

# パターン構造

```mermaid
graph TD

A[意見差] --> B[内集団強化]
B --> C[外集団敵視]
C --> D[対立拡大]
D --> E[分極化]
```

---

# 特徴

- 中間層が弱まる
- 対話が難しくなる
- アイデンティティ対立化する
