# Hypothesis

Hypothesis は Pattern と Mechanism の仮説を記述するノートである。

Hypothesis は

Research Question
↓
Case
↓
Comparison
↓
Pattern

の後に生成される。

---

# 基本構造

```mermaid
flowchart TD

ResearchQuestion --> Case

Case --> Comparison

Comparison --> Pattern

Pattern --> Hypothesis

Hypothesis --> Mechanism
```
