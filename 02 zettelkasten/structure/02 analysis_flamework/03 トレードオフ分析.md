# 概要

Tradeoff Analysisは、複数の目標が同時に最大化できない場合に、最適なバランスを決定する分析フレームワークである。
多くの意思決定は、最適解ではなく複数の価値のバランスを選ぶ問題である。
# Tradeoffの基本構造

```mermaid
flowchart TD

A[Decision]
B[Option A]
C[Option B]

D[Cost]
E[Quality]
F[Speed]

A --> B
A --> C

B --> D
B --> E
B --> F

C --> D
C --> E
C --> F
```
意思決定ではすべての評価軸を同時に最大化できない。
# 手順
```mermaid
flowchart TD 
    A[目的を定義] --> B[評価軸を決める]
    B --> C[選択肢を列挙]
    C --> D[各軸で評価]
    D --> E[バランスを選ぶ]
```
# 重要性
多くの失敗は、Tradeoffを無視することで起きる。
- すべてを最大化しようとする
- 制約を認識しない
