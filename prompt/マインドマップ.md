# Output Format: 3階層ルールによる構造化
以下の形式で、章ごと、または論理ブロックごとに整理してください。

## 1. 【全体地図：3階層アウトライン】
マインドマップの設計図として、以下の3層で整理してください。
- 第1階層（太い枝）：章のタイトルや主要テーマ
- 第2階層（中くらいの枝）：核心となる「概念」や「ロジックの節目」
- 第3階層（細い枝）：RAG（検索）にヒットすべき具体的な「キーワード」や「専門用語」

## 2. 【動的プロセス：状態遷移の予兆】
本書の中で「状態の変化」「時系列のプロセス」「因果関係のループ」が含まれる箇所を特定し、以下の形式でリストアップしてください。
- 箇所：どのセクションか
- 状態A → 状態B：変化の概要
- 遷移のトリガー：変化を起こす原因や条件

## 3. 【速読のための「問い（Query）」】
私が原文を検証する際に意識すべき、鋭い問いを3つ提示してください。

# Constraints
- 網羅性よりも「構造（ロジック）」を優先してください。
- 後でMermaid形式（Flowchart）に変換しやすいよう、名詞（状態）と動詞（遷移条件）を明確に区別して記述してください。
